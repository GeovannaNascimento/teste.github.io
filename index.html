Navegadores antigos tinham uma frase gloriosa: "Navegar é preciso; viver não é preciso." Quero para mim o espírito desta frase, transformada a forma para a casar com o que eu sou: Viver não é necessário; o que é necessário é criar.

O comando <html>, é utilizado para estruturar a página.

O <head>, serve para declarar informações, tais como o título por exemplo.

Para iniciarmos um parágrafo, usa-se <p>.

Quando quisermos inserir uma representação tipográfica padrão no tipo itálico, devemos utilizar o comando <i>

Para criar intervalos de texto, digitamos <b> .

E sempre que quisermos encerrar um código, utilizamos uma /, como por exemplo: </html>,</head>,</p>,</i> e </b>
